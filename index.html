Nos services

Deux facons de terminer le meme parcours KYC

Le design reprend l'esprit du repo `Bot-Baybridge` avec une page de marque tres lisible et un widget flottant toujours disponible.

Parcours manuel

L'utilisateur envoie ses documents, suit les controles, verifie les champs extraits puis soumet son dossier sans passer par le chatbot.

Assistant en surimpression

Le widget flottant ouvre une conversation proche du repo cible et permet d'envoyer un document directement depuis le chat.

Remplissage automatique

Quand un document est accepte, les informations extraites alimentent automatiquement le formulaire KYC a droite.

Parcours utilisateur

Completer la verification sans le chat

Chaque document passe par le meme moteur de validation. Les champs du formulaire sont mis a jour des qu'une extraction reussit.

  1. 1
    Demarrer le dossier

    Preparer une piece d'identite ou un justificatif de domicile.

  2. 2
    Envoyer les documents

    Televerser un fichier a la fois pour verification.

  3. 3
    Verifier les donnees extraites

    Corriger si necessaire avant la soumission finale.

  4. 4
    Soumettre la verification

    Envoyer le dossier complet pour controle final.
